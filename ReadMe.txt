// 概要

  コマンドプロンプトウィンドウを使い易くしたソフトです。

  コマンドプロンプトを隠しておいて、画面表示や キー操作など
  ユーザーインターフェース部分だけを置き換えています。

  Windows2000/XPで動作する Win32アプリケーションです。

  使い方などは配布元
  http://www.geocities.jp/cygwin_ck/
  を参照してください。

  詳細は http://www.gnu.org/licenses/gpl.txt
  を参照してください。


////////////////////////////////////////////
// コンパイル

  コンパイルするには
   Microsoft Visual C++ Toolkit 2003
   Microsoft Platform SDK
   cygwin
  が必要です。
  環境構築しておき、nmake でコンパイルできます。
